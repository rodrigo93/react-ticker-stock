# Getting Started üöÄ

This project is using:

- React v18
- Node v20
- Ruby on Rails v7
- Ruby v3
- RSpec

## Setup üõ†

### Backend / API

1. Navigate to the `api` directory
2. Run `bundle install`
3. Run `rails db:create`
4. Run `rails db:migrate`
5. Run `rails s`
6. Navigate to `http://localhost:3000`

```shell
cd api  
bundle install
rails db:create
rails db:migrate
rails s
```

### Frontend

1. Navigate to the `frontend` directory
2. Run `npm install`
3. Run `npm start`
4. Navigate to `http://localhost:3001`

```shell
cd frontend
npm install
npm start
```
